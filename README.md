# Month Money Manager v2
MMM, a budget manager，用于计算月度每日预算，方便用户进行开销控制，预防剁手。
v2在[原版](https://github.com/Hochikong/MonthMoneyManager)的基础上修复了数个bug并带来了数个全新功能，
包括资产记录和待摊费用等功能，并基于ElementUI实现了一套全新的UI和复古的配色方案，想体验就请直接下载release里的安装包吧。


## Compile & Installation
from source code = git clone => npm install => electron-forge make => install it!

from prebuilt package = download from releases => unzip or install it!
